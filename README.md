# MOMO
## 모모
**모두모여** 의 줄임말로 사용자가 좀 더 편하고 손쉽게 **팀빌딩** 서비스를 이용할 수 있도록 만든 사이트 
- 팀 빌딩(Team Building) : TEAM(조직)과 BUILDING(만들기)의 합성어

- 개발 목표 :
다양한 분야의 팀빌딩을 위한 팀빌딩 플랫폼 서비스 제작
